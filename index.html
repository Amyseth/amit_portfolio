Case Study

Hotel Management SaaS

Client: Grand Resort & Suites
Duration: 4 Months
Services: Full-Stack SaaS / Grid UX
Tech Stack: HTML5, CSS Grid, Vanilla JS, LocalStorage

The Challenge

The client struggled with slow offline check-in books, unscheduled double room bookings, and zero real-time statistics on seasonal occupancy levels. They required an interactive, instant booking interface that room managers could control efficiently.

The Solution

We developed a modern SaaS application displaying a live-updating grid of hotel occupancy levels. Features included:

  • Live Interactive Room Grid: Instantly check-in or reserve rooms across multiple floors by clicking visual cells.
  • Intelligent Occupancy Status Indicator: Interactive cards updating real-time ratios (Vacant, Occupied, Cleaning).
  • Detailed Financial Analytics: Visual widgets tracking gross earnings, cleaning timelines, and average room revenue.
  • Integrated Local DB System: Keeping operations offline-first so that network slowdowns never block physical check-ins.

Impact & Metrics

35% Check-in Speedup
0% Double Booking Errors
99.9% Operational Uptime

Solar Sales Accelerator

Project Status: Under Deployed Beta Testing
Objective: 3x Solar Lead Closures
Services: Full-Stack AI Integration / ROI Algorithmic math
Tech Stack: Next.js App Router, TailwindCSS, FastAPI, Python, Gemini API

The Challenge & Goals

Assessing solar yield efficiency and explaining returns on investment (ROI) is a major friction point for solar sales reps. Standard quotes take hours to compute and lack emotional sales arguments. Our goal is to build an instant proposal accelerator that calculates exact panel needs, financial payback timelines, and generates a personalized, highly persuasive AI sales pitch tailored directly to the prospect's profile.

Core System Architecture

The deployed Next.js web client interfaces with a Python FastAPI server containing proprietary solar estimation formulas, featuring:

  • ROI Metric Engines: Computes panel dimensions, total installation pricing, and yearly paybacks based on monthly bills.
  • Gemini AI Pitch Builder: Integrates custom LLM prompts to synthesize highly targeted sales presentations, including objection handling and financial benefits.
  • Responsive Lead Vault: Capture name, phone, roof square-footage, and bills, displaying visual result metrics with clean Framer Motion fades.
  • One-Click Clipboard: Direct exports allowing solar reps to instantly copy and text/email pitches directly to customers.

Core Target Metrics

3x Proposal Delivery Speed
<2s AI Response Latency
95% Target Lead Engagement
Case Study

OptiMail RL AI Engine

Project: AI Research & Deployment
Duration: 3 Months
Services: AI Reinforcement Loop / RESTful Dashboard
Tech Stack: Python, Q-Learning, Node.js API, HTML5/CSS3

The Challenge

Email marketing campaigns often suffer from low click-through rates (CTR) due to static subject lines and timing. Standard A/B testing is slow. We wanted to build a deployed reinforcement learning engine that learns client click behaviors and refines subject lines dynamically.

The Solution

We built a fully functional reinforcement learning (RL) campaign suite consisting of:

  • Deployed Q-Learning Engine: An intelligent agent adjusting subject line sentiment coefficients based on direct customer email opens and clicks.
  • Live Performance Dashboard: A visual terminal displaying ongoing CTR metrics, reward curves, and sentiment vectors.
  • Subject Line Multi-generator: An interactive UI letting users input keywords and see AI-scored CTR variations.
  • Webhook Integrations: Real-time hooks capturing simulated clicks to train the RL agent in the background.

Impact & Metrics

+42% Click-Through Rate Boost
50ms API Decision Time
3x Faster Campaign Adapts
